Introduction

This is a Computer Networks course project with the objective of building a web application that allows to remotely control computers within a local area network (LAN), Inspired by TeamViewer, the application is designed with a Christmas theme to create a more vivid and memorable user interface.

Main Functionalities of the Backend and Agent (Controlled Machine):

  • Enable/disable the Webcam on the server machine, record video, and stream it to the client.
  • Keylogger :
    • macOS : Capture keystrokes on the server machine via a local web page (/server-keycapture) and relay them to the client in real time (not a system-wide keylogger).
    • Windows : Capture all keystrokes system-wide.
  • Capture the screen : remotely and send it to the web client.
  • Send Restart and Shutdown commands to machines within the LAN.
  • Establish real-time connections using WebSocket / Socket.
