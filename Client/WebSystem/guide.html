Usage Instruction

1. Prepare

  • All machines must be connected to the same Local Area Network (LAN).
  • Follow the instructions in the README.md file.

2. Connect

  • Open the main web page of the project.
  • Enter the IP address of the server / WebSocket.
  • Click the Connect button to establish a real-time connection.
  • Log in using an existing username and password, or enter the Master Code to create a new account.

3. Key Web Features

  • Camera: view the live video stream from the webcam of the controlled machine.
  • Screen: request a screen capture and display the latest captured image.
  • Keylogger: Start/stop the keylogger, view the list of keystrokes recorded by the agent, and clear the captured logs.
  • Power: send remote Restart or Shutdown commands to the machine.
